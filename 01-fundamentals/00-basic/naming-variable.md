# Naming Variable

## Variable
**Makna:** Suatu tempat atau kotak penyimpanan suatu nilai yg akan digunakan oleh program di deklarasi dan di inisialisasi dengan tipe data tertentu, penyimpanan bersifat sementara karna di simpan di dalam ram komputer.
## Naming
**Makna:** Suatu konsep/format pemberian nama kepada variable, class dan method. Penamaan yg baik harus memberikan arti kepada variable class maupun method agar memudahkan dalam membaca code, format yg digunakan untuk pemberian nama pada class dimulai huruf depan dengan kapital, lalu penamaan variable dan method menggunakan konsep camelCase yaitu kata huruf depan dengan huruf kecil dilanjutkan kata kedua di awal huruf kapital. Penamaan variable boolean biasa memiliki di depan kata nya (**was, is, has**).
